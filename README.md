# Клиент для Trello API

## [Демо](https://isvk.github.io/trello/build/#/)

## Реализованный функционал

* Авторизация
* Выбор досок
* CRUD для карточек
* Сортировка и перенос(drag-and-drop) для листов и карточек.

## Стек

* React
* Hooks
* Redux
* Saga
* Reselect
* Immutable.js
* React Router
* SCSS
* React-beautiful-dnd
* REST API

## Back-End

Рабочий сервер Trello API
